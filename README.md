# JavaScript Basics Used in These Programs

The programs use **JavaScript fundamentals** such as:

- **Variables** – Used to store data values.
- **Data Types** – Represent different kinds of data like numbers, strings, and booleans.
- **Operators** – Perform operations such as addition, subtraction, comparison, and modulo.
- **Conditional Statements (`if-else`)** – Used to make decisions based on conditions.
- **Functions** – Reusable blocks of code that perform specific tasks.
- **Arrow Functions** – A shorter syntax for writing functions.
- **Loops (`for`, `while`)** – Used to repeat a block of code multiple times.
- **Nested Loops** – Loops placed inside other loops for complex repetitive tasks.
- **Mathematical Operations** – Used for calculations and number manipulation.

These concepts are applied to solve problems such as:

- Checking whether a number is **Even or Odd**
- Evaluating **Pass or Fail** results
- Finding the **Largest of Three Numbers**
- Calculating the **Sum from 1 to N**
- Generating a **Multiplication Table**
- Counting the **Number of Digits**
- Reversing a **Number**
- Calculating a **Factorial**
- Checking whether a number is **Prime**
- Creating **Star Patterns** using loops
